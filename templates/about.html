About BITS Confessions Bot

What is this?

BITS Confessions Bot is a web application that allows you to search through anonymous confessions and discussions from BITS Pilani students across various Reddit forums. The application collects posts and comments from subreddits like r/BITSPilani, r/Indian_Academia, and similar communities, processes them, and makes them searchable.

How does it work?

The application uses the following technologies and approaches:

  1. Data Collection: We scrape public Reddit forums related to BITS Pilani using PRAW (Python Reddit API Wrapper).
  2. Data Processing: The collected posts and comments are processed using natural language processing techniques to categorize them and make them searchable.
  3. Search Functionality: When you enter a query, the application uses text similarity algorithms to find the most relevant confessions and displays them to you.

Privacy and Ethics

This application only collects publicly available data from Reddit. We do not:

  • Store or collect any personal information about users
  • Modify or alter the original content in any significant way
  • Share or sell any data to third parties

All data displayed in this application is already publicly accessible on Reddit. The application simply makes it easier to search and categorize this information.

Disclaimer

This application was created for educational purposes. The confessions and opinions expressed in the posts do not represent the views of BITS Pilani or the developers of this application. Use the information at your own discretion.

Technical Details

Tech Stack

  • Backend: Flask (Python)
  • Database: SQLite / SQLAlchemy
  • Reddit API: PRAW (Python Reddit API Wrapper)
  • Text Processing: NLTK, scikit-learn
  • Data Analysis: pandas
  • Frontend: HTML, CSS, JavaScript, Bootstrap

Key Features

  • Natural language search for confessions
  • Category-based filtering
  • Automatic categorization of posts
  • Display of original Reddit sources
  • Responsive design for all devices
